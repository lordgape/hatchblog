
# Description

[Hatchblog](https://github.com/lordgape/hatchblog) Blog post Integration using hatchway.
![Tests](https://github.com/lordgape/hatchblog/actions/workflows/ci.yml/badge.svg)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run dev
```

## Test

```bash
# unit tests
$ npm run test
```
